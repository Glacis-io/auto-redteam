# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in autoredteam, please report it responsibly.

**Email:** security@glacis.io

**Do not** open a public GitHub issue for security vulnerabilities.

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment:** within 48 hours
- **Initial assessment:** within 5 business days
- **Fix or mitigation:** as soon as practical, coordinated with reporter

## Scope

This policy covers the autoredteam open-source tool. For vulnerabilities in the Glacis paid service, contact security@glacis.io with "Glacis Service" in the subject line.

## Supported Versions

| Version | Supported |
|---|---|
| 0.3.x | Yes |
| < 0.3 | No |
